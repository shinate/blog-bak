下载及安装

[bash]
git clone git://github.com/joyent/node.git
cd node
./configure --prefix=/servers/app/node
make && make install
[/bash]

启动

[bash]
/servers/app/node/bin/node
[/bash]

直接执行./node将进入nodejs的命令行
